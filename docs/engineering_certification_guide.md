## Role: Cloud Engineer

### Description

Cloud Engineers design, deploy, and maintain cloud-based infrastructure and services. They are responsible for ensuring high availability, cost optimization, and security compliance across all cloud workloads.

### Required Foundational Skills

- Networking fundamentals (DNS, load balancing, VPNs, subnets)
- Linux and Windows server administration
- Identity and access management (IAM) concepts
- Infrastructure-as-Code (Terraform, Bicep, or ARM templates)
- Monitoring and observability principles
- Cost management and resource tagging strategies

### Required Certifications

| Priority | Certification | Exam Code | Focus Area |
|----------|--------------|-----------|------------|
| Primary | Microsoft Azure Developer Associate | AZ-204 | App services, functions, SDKs |
| Primary | Microsoft Azure Solutions Architect Expert | AZ-305 | Architecture design and governance |
| Secondary | Microsoft Azure Security Engineer | AZ-500 | Security posture and threat protection |
| Secondary | Microsoft Azure Administrator | AZ-104 | Core infrastructure management |

### Recommended Study Pattern

| Parameter | Guideline |
|-----------|-----------|
| Daily focus hours | 1.5 – 2 hours |
| Weekly assessment checkpoints | 1 timed practice exam (Saturday or Sunday) |
| Target practice score before sitting exam | ≥ 82% on three consecutive practice tests |
| Recommended prep duration | 6 – 8 weeks per certification |
| Lab practice | Minimum 3 hands-on labs per week |

## Role: DevOps Engineer

### Description

DevOps Engineers own the CI/CD pipelines, release orchestration, and platform reliability for engineering teams. They bridge development and operations by automating build, test, and deployment workflows.

### Required Foundational Skills

- Source control systems (Git branching strategies, pull request workflows)
- Containerization and orchestration (Docker, Kubernetes)
- CI/CD pipeline design (YAML-based pipelines, multi-stage deployments)
- Scripting and automation (Bash, PowerShell, Python)
- Configuration management and secrets handling
- Site Reliability Engineering (SRE) principles (SLIs, SLOs, error budgets)

### Required Certifications

| Priority | Certification | Exam Code | Focus Area |
|----------|--------------|-----------|------------|
| Primary | Microsoft DevOps Engineer Expert | AZ-400 | CI/CD, source control, monitoring |
| Secondary | Microsoft Azure Administrator | AZ-104 | Prerequisite infrastructure knowledge |
| Secondary | Microsoft Azure Developer Associate | AZ-204 | App services, SDKs, messaging |

### Recommended Study Pattern

| Parameter | Guideline |
|-----------|-----------|
| Daily focus hours | 1.5 hours |
| Weekly assessment checkpoints | 1 topic quiz (midweek) + 1 full practice exam (weekend) |
| Target practice score before sitting exam | ≥ 80% on three consecutive practice tests |
| Recommended prep duration | 5 – 7 weeks per certification |
| Lab practice | Minimum 4 pipeline-focused labs per week |

## Role: Data Engineer

### Description

Data Engineers design and implement data ingestion, transformation, and storage solutions. They build and maintain data pipelines that feed analytics platforms, machine learning models, and business intelligence dashboards.

### Required Foundational Skills

- Relational and non-relational database systems
- Data modeling (star schema, snowflake schema, data vault)
- ETL/ELT pipeline design and orchestration
- Distributed computing fundamentals (Spark, MapReduce)
- Data governance and cataloging
- SQL proficiency (advanced querying, performance tuning)
- Streaming data concepts (event hubs, Kafka-style architectures)

### Required Certifications

| Priority | Certification | Exam Code | Focus Area |
|----------|--------------|-----------|------------|
| Primary | Microsoft Azure Data Engineer Associate | DP-203 | Data pipelines, storage, batch/stream processing |
| Primary | Microsoft Azure Database Administrator | DP-300 | Database management and optimization |
| Secondary | Microsoft Azure Data Scientist Associate | DP-100 | ML integration and model operationalization |

### Recommended Study Pattern

| Parameter | Guideline |
|-----------|-----------|
| Daily focus hours | 1.5 – 2 hours |
| Weekly assessment checkpoints | 1 scenario-based quiz (Wednesday) + 1 full practice exam (Saturday) |
| Target practice score before sitting exam | ≥ 85% on three consecutive practice tests |
| Recommended prep duration | 7 – 9 weeks per certification |
| Lab practice | Minimum 3 data pipeline labs per week |

## Role: Security Engineer

### Description

Security Engineers protect cloud workloads and ensure compliance with organizational and regulatory security standards. They implement threat detection, identity governance, and incident response capabilities.

### Required Foundational Skills

- Identity and access management (Azure AD, RBAC, Conditional Access)
- Network security (NSGs, firewalls, DDoS protection)
- Threat detection and incident response
- Compliance frameworks and audit readiness
- Key management and secrets handling (Key Vault)

### Required Certifications

| Priority | Certification | Exam Code | Focus Area |
|----------|--------------|-----------|------------|
| Primary | Microsoft Azure Security Engineer | AZ-500 | Identity, platform protection, security ops |
| Secondary | Microsoft Security Operations Analyst | SC-200 | Sentinel, Defender, incident response |
| Secondary | Microsoft Azure Administrator | AZ-104 | Infrastructure prerequisite |

### Recommended Study Pattern

| Parameter | Guideline |
|-----------|-----------|
| Daily focus hours | 1.5 hours |
| Weekly assessment checkpoints | 1 full practice exam (weekend) |
| Target practice score before sitting exam | ≥ 82% on three consecutive practice tests |
| Recommended prep duration | 6 – 7 weeks per certification |
| Lab practice | Minimum 3 security-focused labs per week |

## General Study Guidelines

1. **Block calendar time.** Reserve focus hours as recurring calendar events. Treat them as non-negotiable.
2. **Use spaced repetition.** Review missed questions from practice exams at 1-day, 3-day, and 7-day intervals.
3. **Prioritize hands-on labs.** Exam questions increasingly test practical skills over rote memorization.
4. **Join a study cohort.** Teams that study together report 18% higher first-attempt pass rates (internal data, Q1 2026).
5. **Escalate blockers early.** If workload conflicts prevent consistent study, notify your manager within the first two weeks.
